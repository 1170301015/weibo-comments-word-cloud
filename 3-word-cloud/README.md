## 运行所需文件

- 请将需要生成词云的清洗过的文本数据储存于“comment.txt”文件中。
- font_path='xxx.ttf' 是中文字体文件，此目录下提供'SourceHanSans-Bold.ttf'和'SourceHanSans-normal.ttf'两个选择，可以自由更换成其它的中文字体样式。
